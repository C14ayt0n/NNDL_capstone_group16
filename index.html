Project overview

NutritionAI is a browser-based tool that helps users estimate the nutritional content of meals using a model trained on the Nutrition5k dataset.

  • Client-side prediction with TensorFlow.js.
  • Calories and macronutrients estimated from a single RGB photo.
  • Lightweight model exported from a custom CNN.

Model & data

The model is trained on labeled Nutrition5k images with dish-level calories, fat, carbs, and protein. It is converted to TensorFlow.js for fast in-browser inference without a backend.
